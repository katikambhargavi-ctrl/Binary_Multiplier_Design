# 4-bit Braun Multiplier

## Overview

This project implements a **4-bit Braun Multiplier** using Verilog HDL. The design generates partial products using AND gates and combines them with **Half Adders (HA)** and **Full Adders (FA)** to produce an 8-bit multiplication result. The Braun multiplier is a combinational array multiplier that provides a simple and efficient hardware architecture for unsigned binary multiplication.

## Features

* Implements a **4-bit Braun (Array) Multiplier**.
* Multiplies two 4-bit unsigned binary inputs (`A` and `B`).
* Produces an 8-bit product (`P`).
* Uses AND gates for partial product generation.
* Uses Half Adders and Full Adders to sum partial products.
